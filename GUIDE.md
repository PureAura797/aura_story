# Руководство для владельца сайта PureAura

> Простая инструкция без технических терминов

---

## Что умеет ваш сайт

Сайт работает полностью автономно. Через **админ-панель** (закрытая часть сайта) вы можете менять всё сами — без программиста:

| Что можно делать | Где это |
|-----------------|---------|
| Менять тексты на сайте | Контент |
| Менять цены в калькуляторе | Калькулятор |
| Добавлять/удалять отзывы | Отзывы |
| Менять фото команды и портфолио | Медиа |
| Менять телефон, email, мессенджеры | Контакты |
| Настроить куда приходят заявки | Уведомления |
| Подключить рекламные счётчики | Аналитика |
| Сменить пароль, email для восстановления | Настройки |
| Прочитать инструкции и FAQ | Справка |

---

## Как войти в админку

1. Откройте в браузере: **ваш-сайт.ru/admin**
2. Введите пароль (его знает деплойщик, который ставил сайт)
3. Готово — вы внутри

---

## Как менять тексты на сайте

1. В меню слева нажмите **«Контент»**
2. Найдите нужное поле (все подписаны по-русски)
3. Измените текст
4. Нажмите **«Сохранить»**
5. Обновите главную страницу — изменения уже там

---

## Как менять цены

1. Нажмите **«Калькулятор»** в меню
2. Измените базовую цену или коэффициенты
3. Сохраните — калькулятор на сайте сразу обновится

---

## Как менять контакты (телефон, email, мессенджеры)

1. Нажмите **«Контакты»** в меню
2. Измените нужное поле
3. Сохраните — телефон обновится **везде** на сайте: в кнопках, в подвале, в навигации, в виджете мессенджера

---

## Куда приходят заявки с сайта

Когда посетитель заполняет форму на сайте, заявка может прийти:

- **На почту** — как обычное письмо
- **В Telegram** — через бота в чат
- **В MAX** — аналогично через бота
- **В вашу CRM** — через Webhook (настраивает IT-специалист)

Всё это включается в разделе **«Уведомления»**. Можно включить одновременно несколько каналов.

---

## Как забыли пароль?

1. На странице входа нажмите **«Забыли пароль?»**
2. Введите email (тот, что указан в Настройки → Профиль)
3. Получите 6-значный код
4. Введите его → Задайте новый пароль

> Если код не приходит на почту — попросите деплойщика посмотреть код в терминале сервера (он там отображается).

---

## Что передать рекламщику

Рекламщику нужно дать:
1. Адрес сайта + пароль от админки
2. Сказать ему зайти в **Аналитика**
3. Он сам вставит нужные ID счётчиков (Яндекс.Метрика, Google, ВКонтакте)
4. Нажмёт «Сохранить» — всё заработает

**Рекламщику не нужен доступ к коду сайта.** Всё делается через админку.

---

## Что передать деплойщику

Деплойщик — это человек, который ставит сайт на сервер и привязывает домен. Ему нужно:

1. **Папку с проектом** (весь код)
2. **Файл README.md** — там техническая инструкция для него
3. **Домен** (например pureaura.ru) — если уже куплен
4. **Хостинг** — сервер, где будет работать сайт

Деплойщик сделает:
- Установит проект на сервер
- Настроит домен
- Создаст файл с паролем
- Запустит сайт

После этого сайт будет доступен по вашему домену.

---

## Работает ли админка с телефона?

**Да!** Можно заходить в админку с любого устройства — телефона, планшета, компьютера. Меню адаптируется под размер экрана.

## Словарь (если деплойщик или рекламщик говорят непонятные слова)

| Слово | Что это простыми словами |
|-------|------------------------|
| **Домен** | Адрес сайта (pureaura.ru) |
| **Хостинг** | Компьютер в интернете, на котором работает сайт |
| **SMTP** | Настройки почтового сервера для отправки писем |
| **Метрика** | Счётчик от Яндекса — показывает сколько человек зашло на сайт |
| **GA4** | Счётчик от Google — аналог Метрики |
| **GTM** | «Менеджер тегов» от Google — удобный способ добавлять счётчики |
| **VK Pixel** | Код от ВКонтакте для ретаргетинга (показ рекламы тем, кто был на сайте) |
| **Telegram-бот** | Робот в Телеграме, который пересылает заявки с сайта в ваш чат |
| **Webhook** | Ссылка, по которой сайт автоматически отправляет данные в другую систему |
| **SSL** | Замочек в адресной строке (означает безопасное соединение) |
